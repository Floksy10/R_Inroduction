1 Write your own my_cumsum() function

The base R function cumsum() returns the cumulative sum of its argument, e.g.

cumsum(1:4)
[1]  1  3  6 10
  1. Write the function my_cumsum() that does the same.
my_cumsum <- function(x) {
  
    sum <- cumsum(x)
    
  return(sum)
  }
  1. Show that the function works for the argument -1:4
my_cumsum(-1:4)

2 Bootstrap

  1. Source the bootstrap() function of the lecture slides.
bootstrap <- function(formula, data, B = 1000) {  
  
  pars   <- lm(formula = formula, data = data) %>% 
    coef() %>% 
    names()
  
  output <- matrix(nrow = B, ncol = length(pars), 
                     dimnames = list(NULL, pars))
  
  for (i in 1:B) {
    
    bootsample  <- slice_sample(.data = data, n = nrow(data), replace = TRUE)
    
    bootfit     <- lm(formula = formula, data = bootsample)
    
    output[i, ] <- bootfit$coef
  }
  
  return(output)
}

  1. In a 2 x 2 plot array, plot the bootstrap sampling distributions of the four parameters in red, and the theoretical sampling distribution of the parameters in blue. For the latter, use stat_function(fun = dnorm, args = list(mean = ..., sd = ...)), using the mean and sd values of the corresponding lm model estimates.
# Ensure ggplot2 is installed and loaded
if (!requireNamespace("ggplot2", quietly = TRUE)) install.packages("ggplot2")
library(ggplot2)

# Prepare the plot for each parameter
plots <- lapply(1:ncol(bootstrap_ci_df), function(i) {
  # Compute the mean and standard deviation for the theoretical distribution
  param_mean <- mean(bootstrap_ci_df[[i]])
  param_sd <- sd(bootstrap_ci_df[[i]])
  
  ggplot() +
    # Add the bootstrap distribution in red
    geom_density(data = data.frame(Value = bootstrap_ci_df[[i]]), aes(x = Value), fill = "red", color = "red", alpha = 0.5) +
    # Add the theoretical normal distribution in blue
    stat_function(fun = dnorm, args = list(mean = param_mean, sd = param_sd), color = "blue") +
    labs(title = colnames(bootstrap_ci_df)[i]) +
    theme_minimal()
})

# If you have the gridExtra package installed, you can arrange the plots in a 2x2 grid
if (!requireNamespace("gridExtra", quietly = TRUE)) install.packages("gridExtra")
library(gridExtra)
grid.arrange(grobs = plots, ncol = 2)
